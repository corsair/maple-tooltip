Maple Tooltip v1.0.0

Easily include information from MapleStory on your website, blog, etc.

Getting Started

1. Include the Javascript /src/maple.tooltip.js and CSS /css/maple.tooltip.css files in your document.

2. Initialize Maple Tooltip by adding the following before your </body>:

<script>
$('[maple-id]').mapleTooltip({
    region: "gms"
});
</script>

3. Add maple-id="100100" to any element (this can be any monster ID).

4. Congrats, now you have one grand ole' .
